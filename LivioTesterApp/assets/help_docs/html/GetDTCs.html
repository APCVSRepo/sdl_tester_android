Get DTCs

The GetDTCs command allows developers to read diagnostic trouble codes (DTCs) from various modules in the vehicle.

In order to retrieve DTCs from a particular module, the application must know the module's specific CAN bus ID and input this ID as a parameter.

ECU ID (required)

Type: ID (integer)

Min value: 0

Max value: 65,535


The CAN bus ID of the module to read DTCs from.

DTC Mask (optional)

Type: Integer

Min value: 0

Max value: 255


The DTC mask byte, which allows filtering of results to retrieve current DTC status, history status, etc.
