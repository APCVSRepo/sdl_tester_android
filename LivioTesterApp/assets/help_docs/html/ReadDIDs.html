Read DIDs

The ReadDIDs command allows developers to read specific data (DIDs) from various modules in the vehicle.

In order to retrieve DTCs from a particular module, the application must know the module's specific CAN bus ID and the ID of the DID to retrieve.

ECU ID (required)

Type: ID (integer)

Min value: 0

Max value: 65,535


The CAN bus ID of the module to read DIDs from.

DID ID (required)

Type: List<Integer>

Min array size: 1

Max array size: 1,000

Min value: 0

Max value: 65,535


The DID ID for the module to read DIDs from.
