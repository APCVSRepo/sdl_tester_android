Scrollable Message

The ScrollableMessage command allows developers to display a large portion of text to the user.

The following parameters are available to display a long, scrollable message to the user.

Message Body (required)

Type: String

Max value: 500


The body of the message to display. The message can contain line breaks and tab characters.

Timeout (required)

Type: Integer

Min value: 1,000 (ms)

Max value: 65,535 (ms)

Default value: 30,000


This parameter represents the timeout for the scrollable message in milliseconds. When the timeout expires, the scrollable message will be dismissed.

Soft Buttons (optional)

Type: Vector<SoftButton>

Min array size: 0

Max array size: 8


This parameter allows your application to add custom soft buttons to the scrollable message interface. Soft buttons represent additional user actions that are available for the scrollable message. Depending on the HMI, soft buttons could be along-side the scrollable message or in a separate menu available to the user.
