Perform Interaction

The PerformInteraction command allows developers to show a pop-up list dialog that has been added through the CreateInteractionChoiceSet command. If the user selects one of the items in the interaction choice set, the result will be returned to the onPerformInteractionResponse method in the SDL Service. From there, the click can be dispatched to the appropriate click-listener based on the id that was selected. The user also has the option to back out of the menu, in which case the response will show an ABORTED message.

The PerformInteraction command has capabilities for the following parameters:

Initial Text (required)

Type: String

Max length: 500


The interaction title that will show above the choices in the pop-up dialog. This should be a concise description of what the pop-up is asking for.

Interaction Mode (required)

Type: InteractionMode enum


The interaction mode parameter specifies how the user can perform the interaction.

Available Interaction Mode Options

Enumerated Value Interaction Mode
MANUAL_ONLY Only click events are permitted
VOICE_REC_ONLY Only voice events are permitted
BOTH Both click & voice events are permitted

Choice Set IDs (required)

Type: List<Integer>

Min array size: 1

Max array size: 100

Min value: 0

Max value: 2,000,000,000


In order to perform an interaction with the user, the developer must select a set of choice set IDs that have been added to the system using the CreateInteractionChoiceSet command.

Interaction Timeout (optional)

Type: Integer

Min value: 5,000 (ms)

Max value: 10,000 (ms)

Default value: 10,000 (ms)


This value represents how long the pop-up should be displayed to the user. The duration parameter is an integer representing how many milliseconds the pop-up should be displayed for. If this value is omitted, the pop-up will remain on the screen indefinitely - until the app is disconnected or until the user navigates away from it.

Initial Prompt (optional)

Type: List<TTSChunk>

Min array size: 1

Max array size: 100


The interaction voice prompt that will speak the input text to the user when the pop-up dialog shows.

Help Prompt (optional)

Type: List<TTSChunk>

Min array size: 1

Max array size: 100


The help voice prompt that will speak the input text to the user when the user asks for help while the app is open.

Timeout Prompt (optional)

Type: List<TTSChunk>

Min array size: 1

Max array size: 100


The interaction voice prompt that will speak the input text to the user when the pop-up dialog times out.

Voice-Rec Help (optional)

Type: List<VrHelpItem>

Min array size: 1

Max array size: 100


A list of suggestions to help the user navigate the interaction through voice control.

Interaction Layout (optional)

Type: String


Specifies a different type of layout for the specified interaction.
